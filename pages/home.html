My name is James Filby and I am a computer programmer.

This website is designed to mimic both the look and functionality of an old greenscreen terminal

This site operates entirely through keyboard inputs.

All page links have a number after them. To visit a page type 'GOTO' followed by a space followed by the number, followed by the 'ENTER' key e.g. 'GOTO 1'

To view more detailed help, type 'HELP' from any screen.

To return to the homepage type 'HOME' from any screen.
