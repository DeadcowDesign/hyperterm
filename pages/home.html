Welcome to the future. Welcome to the internet.

If you've ever wanted to use a website that makes you feel like Mr. Robot, or Neo from the Matrix - well, this is the site for you.

Instructions

  • Enter commands into the Command bar at the bottom of the screen followed by the 'Enter' key.
  • To return to the homepage from any screen use the command 'HOME'
  • To visit a page type 'GOTO' followed by the number of the link (e.g. 'GOTO 1').
  • For further help use the command 'HELP'
  • You can also use a mouse - but doesn't that feel like cheating...?
