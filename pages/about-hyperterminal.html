About this website

But why, though?

Because I thought it would be fun to have a website that looked like an old computer, but not just an old computer; a properly old computer. I've seen sites before that have been made to look like old DOS systems (in fact there's an excellent framework called Bootstrap.386), but I've never seen one that looked like a green-screen.

The layout and CSS for the site is dead simple - the colours on screen are based on actual phosphor colours of old CRT monitors (see the 'HELP' page for more information on changing the site colours). It uses the VT323 font from Google, and honestly, that's about it.


When I built the site originally, it was just a standard website, albeit with green-screen aesthetics. I put the little flashing cursor and command bar at the bottom just for a little visual flourish, but that got me thinking:


Can I build a site that behaves like an old dumb-terminal?


Turns out the answer is complex.
