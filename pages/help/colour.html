COLOUR command help

Type 'COLOUR' followed by any colour in this list to change the phosphor (text) colour of the OS:

  • WHITE
  • AMBER
  • LT-AMBER
  • GREEN-1
  • APPLE-II
  • GREEN-2
  • APPLE-IIC
  • GREEN-3

For example: 'COLOUR AMBER' will change the phosphorous to amber.

The default colour is GREEN-2

Fun Facts: The phosphor colours are based on actual phosphor colours
