• HELP - show the help screen
  • LIST - show all the pages on the site
  • open [PAGE] - load a page
  • RESET - reset the terminal
  • CLEAR - empty the terminal window
  • GLOW - toggle screen glow effect
  • SCANLINES - toggle scanlines effect
  • COLOUR [COLOUR] - change phosphor colour

Type 'HELP [ITEM]' for more detailed help on each command.
